# arkanoid_game_react

<i>Break the wall</i> is React arkanoid game

You can play it here : 
https://paulinablaszk.github.io/break_the_wall/

<h3>Technologies used</h3>
 * React.js
 * ES7
 * Babel
 * Webpack
 * Gulp
 * SCSS
 
<h3>Features</h3>
* levels
* lives
* bonuses
    * shorter / longer paddle
    * slower / faster ball
    * extra life
