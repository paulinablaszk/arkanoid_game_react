# arkanoid_game_react

<i>Break the wall</i> is React arkanoid game

You can try it here : 
https://paulinablaszk.github.io/break_the_wall/

Technologies used: React.js, ES7, Babel, Webpack, Gulp, SCSS
 
Features
* levels
* lives
* bonuses
    * shorter / longer paddle
    * slower / faster ball
    * extra life
