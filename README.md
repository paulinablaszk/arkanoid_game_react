# break_the_wall
<i>Break the wall</i> is React arkanoid game
Check it out at [https://paulinablaszk.github.io/break_the_wall/]

<h3>Features</h3>
* levels
* lives
* bonuses
    * shorter / longer paddle
    * slower / faster ball
    * extra life

<h3>Technologies used</h3>
 * React.js
 * ES7
 * JSX
 * Webpack
 * Gulp
 * SCSS
